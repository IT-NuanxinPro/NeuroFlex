## 交互动画

### 展开/收起动画
```
收起 → 展开
┌─────┐     ┌─────┐
│  ▼  │ --> │  ▲  │
└─────┘     └─────┘
  (旋转180度)

内容区域
高度: 0 → 自动
透明度: 0 → 1
动画时长: 0.3s
```

### 悬停效果
```
正常状态          悬停状态
┌─────┐          ┌─────┐
│     │   -->    │     │ (向上移动2px)
└─────┘          └─────┘ (阴影增强)
```

### 点击反馈
```
正常 → 按下 → 释放
100%   95%    100%
(缩放动画)
```

## 颜色方案

### 主题色
- 主色调: `#00d4ff` (青蓝色)
- 次要色: `#7b2cbf` (紫色)
- 成功色: `#00ff88` (绿色)
- 警告色: `#ffaa00` (橙色)
- 错误色: `#ff3366` (红色)

### 背景色
- 主背景: `#0a0a1a` (深蓝黑)
- 卡片背景: `rgba(255, 255, 255, 0.05)` (半透明白)
- 玻璃效果: `backdrop-filter: blur(10px)`

### 文字色
- 主文字: `#ffffff` (白色)
- 次要文字: `rgba(255, 255, 255, 0.6)` (半透明白)
- 强调文字: `#00d4ff` (主题色)

## 响应式断点

- **移动端**: < 768px
- **平板**: 768px - 1023px
- **PC端**: ≥ 1024px

## 字体大小

- **超大**: 32px (分数徽章)

- **超小**: 12px (标签)
